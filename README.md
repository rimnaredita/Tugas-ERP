## Tahapan

-   `$ git clone https://github.com/ibishek/personal-finanace-system.git`
-   `$ cd project-folder`
-   `$ composer install`
-   ubah nama file `.env.example` menjadi `.env`
-   `$ php artisan key:generate`
-   Buat database dengan nama `personal_finance` 
-   `$ php artisan migrate`
-   `$ php artisan db:seed`
-   `$ php artisan serve`
